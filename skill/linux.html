Linux

Saya menggunakan Linux lewat virtual machine dengan Debian 11 sebagai sistem operasinya. Saya memakai Debian 11 karena tampilannya sederhana, ringan, dan cukup nyaman digunakan untuk belajar maupun mencoba berbagai konfigurasi dan perintah Linux. Dengan virtual machine, saya bisa menjalankan Linux tanpa harus menginstalnya langsung di perangkat utama, jadi lebih aman dan fleksibel untuk digunakan..

Di sini saya ingin menunjukkan beberapa hal dasar pada Linux yang biasanya perlu dilakukan terlebih dahulu, salah satunya adalah melakukan konfigurasi alamat IP secara manual. Konfigurasi IP ini penting agar perangkat Linux dapat terhubung ke jaringan maupun internet dengan baik. Langkah pertama yang dilakukan adalah membuka file konfigurasi network menggunakan terminal dengan perintah: ”nano /etc/network/interfaces”

Penjelasannya:

  • auto eth0 digunakan agar interface jaringan otomatis aktif saat Linux dinyalakan.
  • iface eth0 inet static menandakan bahwa IP yang digunakan bersifat static atau tetap.
  • address merupakan alamat IP yang akan digunakan.
  • netmask digunakan untuk menentukan jaringan.
  • gateway adalah jalur utama untuk terhubung ke jaringan lain atau internet.

Setelah konfigurasi selesai dibuat, tekan CTRL + O untuk menyimpan file lalu CTRL + X untuk keluar dari editor nano. Kemudian langkah berikutnya adalah me-restart layanan network agar konfigurasi yang sudah dibuat dapat diterapkan pada sistem Linux. Caranya dengan menggunakan perintah: ”systemctl restart networking” atau “service networking restart” atau juga bisa /etc/init.d/networking restart. Setelah itu cek apakah ip sudah ada atau belum dengan ketik perintah: ”ip a”

Selanjutnya, agar PC client atau perangkat yang digunakan bisa mendapatkan alamat IP secara otomatis, kita dapat menggunakan DHCP Server. DHCP (Dynamic Host Configuration Protocol) berfungsi untuk membagikan alamat IP secara otomatis kepada setiap client yang terhubung ke jaringan, sehingga kita tidak perlu mengatur IP secara manual satu per satu. Pada sistem operasi Linux sebenarnya fitur DHCP Server sudah tersedia, tetapi tidak semua distribusi Linux langsung menyediakan paket DHCP Server secara bawaan. Karena itu, kita perlu menginstalnya terlebih dahulu. Pada Debian 11, paket DHCP Server yang umum digunakan adalah isc-dhcp-server.

Untuk menginstalnya, jalankan perintah berikut di terminal: apt install isc-dhcp-server - untuk menginstall DHCP Server Setelah proses instalasi selesai, kita perlu masuk dan melakukan konfigurasi DHCP Server agar dapat digunakan. Untuk membuka dan mengedit file konfigurasi tersebut, gunakan perintah:

nano /etc/dhcp/dhcpd.conf

Di dalam file tersebut kita bisa mengatur range IP, gateway, DNS, dan pengaturan jaringan lainnya sesuai kebutuhan. Setelah konfigurasi selesai, layanan DHCP Server dapat dijalankan atau direstart menggunakan perintah: "systemctl restart isc-dhcp-server"

Lalu untuk memastikan layanan DHCP berjalan dengan baik, gunakan perintah: "systemctl status isc-dhcp-server"

Dengan menggunakan DHCP Server, proses konfigurasi jaringan menjadi lebih mudah, cepat, dan efisien karena setiap perangkat client akan otomatis mendapatkan alamat IP saat terhubung ke jaringan.
