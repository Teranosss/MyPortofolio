# My-Portofolio

Portfolio website milik **Amarsya Terano** — Network Engineering enthusiast.

🌐 **Live Site:** [teranosss-portofolio.vercel.app](https://amarsyaterano-portofolio.vercel.app)

---

## Tentang

Website portfolio pribadi yang menampilkan:
- **About** — latar belakang dan statistik
- **Experience** — pengalaman kerja dan kegiatan
- **Skills** — keahlian teknis (Mikrotik, Cisco, Linux, Windows, Microsoft Office, Canva)
- **Certificate** — sertifikat yang dimiliki
- **Contact** — informasi kontak

## Teknologi

- HTML5
- CSS3
- JavaScript (Vanilla)

## Cara Menjalankan

Buka `index.html` di browser, atau akses via GitHub Pages.
